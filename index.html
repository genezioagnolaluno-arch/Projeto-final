Atv 01Cálculo de Tensão em um Eixo de Transmissão
▼

Enunciado

Contexto: Um engenheiro precisa verificar se a tensão normal em um eixo circular está dentro dos limites de segurança após a aplicação de uma carga axial.

Código Python

import math

forca = float(input("Força axial (N): "))
diametro = float(input("Diâmetro do eixo (mm): "))

area = math.pi * (diametro / 2)**2
tensao = forca / area

print(f"A tensão normal é de {tensao:.2f} MPa")

Atv 04Dilatação Térmica de uma Tubulação
▼

Enunciado

Contexto: Calcular a variação no comprimento de uma tubulação de vapor de aço inoxidável devido ao aumento de temperatura.

Código Python

l0 = float(input("Comprimento inicial (m): "))
alfa = 17.3e-6 # Aço Inox
ti = float(input("Temperatura inicial (°C): "))
tf = float(input("Temperatura final (°C): "))

delta_l = l0 * alfa * (tf - ti)

print(f"A variação de comprimento é: {delta_l*1000:.2f} mm")

Atv 06Velocidade de Corte em Torneamento
▼

Enunciado

Contexto: Um operador de torno precisa calcular a velocidade de corte (Vc) para ajustar a rotação da máquina.

Código Python

import math

d = float(input("Diâmetro da peça (mm): "))

Atv 07Perda de Carga em Tubulação
▼

Enunciado

Contexto: Calcular a queda de pressão em uma tubulação horizontal para dimensionar uma bomba hidráulica.

Código Python

f = float(input("Fator de atrito: "))
l = float(input("Comprimento (m): "))
d = float(input("Diâmetro interno (m): "))
rho = float(input("Densidade do fluido (kg/m³): "))
v = float(input("Velocidade do fluido (m/s): "))

delta_p = f * (l / d) * (rho * v**2 / 2)

print(f"Queda de pressão: {delta_p:.2f} Pa")
Atv 08Torque em Parafuso de Fixação
▼

Enunciado

Contexto: Determinar o torque necessário para atingir uma força de união específica em um parafuso de fixação.

Código Python

k = 0.2 # Coeficiente de atrito padrão
d = float(input("Diâmetro nominal do parafuso (m): "))

Atv 09Volume de Material em um Fundido
▼

Enunciado

Contexto: Calcular o volume de metal necessário para preencher um molde cilíndrico de fundição com base no raio e altura.

Atv 12Cálculo Estrutural de Treliças Planas
▼

Enunciado

Contexto: Determinar as reações internas em um nó de uma treliça plana isostática com base na carga e inclinação angular.

Código Python

import math

carga = float(input("Carga aplicada no nó (N): "))
angulo_deg = float(input("Ângulo de inclinação do membro (graus): "))

rad = math.radians(angulo_deg)
forca_membro = carga / math.sin(rad)

print(f"Força resultante no membro da treliça: {forca_membro:.2f} N")

Atv 14Mecanismos: Relação de Transmissão por Engrenagens
▼

Enunciado

Contexto: Calcular a razão de engrenamento e a rotação de saída (RPM) para um par de engrenagens cilíndricas retas acopladas.

Código Python

n_entrada = int(input("Número de dentes da engrenagem motriz (entrada): "))
n_saida = int(input("Número de dentes da engrenagem movida (saída): "))
rpm_entrada = float(input("Velocidade de entrada (RPM): "))

i = n_saida / n_entrada
rpm_saida = rpm_entrada / i

print(f"Razão de transmissão (i): {i:.2f}")
print(f"Velocidade angular de saída: {rpm_saida:.2f} RPM")
