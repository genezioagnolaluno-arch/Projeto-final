Projeto 01Cálculo de Tensão em um Eixo de Transmissão
▼

Contexto Técnico:

Um engenheiro precisa verificar se a tensão normal em um eixo circular está dentro dos limites de segurança após a aplicação de uma carga axial.

Código Python Integrado:

import math

forca = float(input("Força axial (N): "))
diametro = float(input("Diâmetro do eixo (mm): "))

area = math.pi * (diametro / 2)**2
tensao = forca / area

print(f"A tensão normal é de {tensao:.2f} MPa")

Relatório 02Avaliação Comparativa de Modelos de Linguagem
▼

Enunciado Oficial:

Redigir uma análise crítica e comparativa dos códigos e gráficos produzidos pelas LLMs (ChatGPT, Copilot e Gemini) focado no comportamento de resolução numérica de estruturas de grande escala, robustez para N=32 e conformidade com a engenharia.

Matriz de Desempenho e Conclusões:

Critério Analítico Copilot Treliças ChatGPT (Trabalho Completo) Gemini Análise Gráfica
Corretude Numérica Aproxima forças internas via deslocamentos (qualitativo). Fiel ao cálculo clássico (cossenos diretores, reações de apoio). Batem com cálculo manual. Correto para exemplo pequeno (N=3), mas não generaliza bem.
Conclusão do Relatório: O ChatGPT apresentou a melhor resposta consolidada e robustez sob malhas de grande escala, enquanto o Gemini gerou um código com lacunas de generalização para N=32.

Projeto 05Dilatação Térmica de uma Tubulação
▼

Contexto Técnico:

Calcular a variação no comprimento de uma tubulação de vapor de aço inoxidável devido ao aumento de temperatura.

Código Python Integrado:

l0 = float(input("Comprimento inicial (m): "))
alfa = 17.3e-6 # Aço Inox
ti = float(input("Temperatura inicial (°C): "))
tf = float(input("Temperatura final (°C): "))

print(f"A variação de comprimento é: {delta_l*1000:.2f} mm")

Projeto 08Perda de Carga em Tubulação
▼

Contexto Técnico:

Calcular a queda de pressão em uma tubulação horizontal para dimensionar uma bomba hidráulica.

Código Python Integrado:

f = float(input("Fator de atrito: "))
l = float(input("Comprimento (m): "))
d = float(input("Diâmetro interno (m): "))
rho = float(input("Densidade do fluido (kg/m³): "))
v = float(input("Velocidade do fluido (m/s): "))

delta_p = f * (l / d) * (rho * v**2 / 2)

print(f"Queda de pressão: {delta_p:.2f} Pa")
Projeto 09Torque em Parafuso de Fixação
▼

Projeto 10Volume de Material em um Fundido

Projeto 14Mecanismos: Relação de Transmissão por Engrenagens
▼

Contexto Técnico:

Calcular a razão de engrenamento e a rotação de saída (RPM) para um par de engrenagens cilíndricas retas acopladas.

Código Python Integrado:

n_entrada = int(input("Número de dentes da engrenagem motriz (entrada): "))
n_saida = int(input("Número de dentes da engrenagem movida (saída): "))
rpm_entrada = float(input("Velocidade de entrada (RPM): "))

i = n_saida / n_entrada
rpm_saida = rpm_entrada / i

print(f"Razão de transmissão (i): {i:.2f}")
print(f"Velocidade angular de saída: {rpm_saida:.2f} RPM")
