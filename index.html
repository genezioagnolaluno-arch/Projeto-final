Atv 01Análise de Tensões Combinadas em Eixos
▼

Enunciado

Determinar as tensões normais e de cisalhamento combinadas em um eixo sob torção e flexão simultâneas, validando o fator de segurança.

Código Python

import math
def verificar_eixo(M, T, d, Sy):
    sigma = (32 * M) / (math.pi * d**3)
    tau = (16 * T) / (math.pi * d**3)
    sigma_vm = math.sqrt(sigma**2 + 3*(tau**2))
    return sigma_vm, Sy / sigma_vm

▼

Enunciado

Código Python

def perda_carga_simples(f, L, D, v, g=9.81):
    return f * (L / D) * (v**2 / (2 * g))

Atv 10Cálculo Estrutural de Treliças Planas
▼

Enunciado

Determinar as forças nos nós de uma treliça isostática elementar utilizando trigonometria e equilíbrio estático.

Código Python

import math
def forca_no_trelica(carga, angulo_deg):
    rad = math.radians(angulo_deg)
    # Somatório de forças verticais
    forca_membro = carga / math.sin(rad)
    return forca_membro
Atv 11Método de Eliminação de Gauss-Jordan
▼

Enunciado

Solução manual de sistemas lineares de equações simultâneas para cálculo de reações de apoio.

Código Python

def gauss_jordan_2x2(A, b):
    # Solução explícita simplificada para sistemas estruturais 2x2
    det = A[0][0]*A[1][1] - A[0][1]*A[1][0]
    if det == 0: return None
    x = (b[0]*A[1][1] - b[1]*A[0][1]) / det
    y = (A[0][0]*b[1] - A[1][0]*b[0]) / det
    return [x, y]

Atv 14Mecanismos: Cálculo de Razão de Transmissão
▼

Enunciado

Calcular a velocidade angular de saída e torque estático em um sistema de engrenagens acopladas.

Código Python

def razao_transmissao(N_entrada, N_saida, rpm_entrada):
    i = N_saida / N_entrada
    rpm_saida = rpm_entrada / i
    return i, rpm_saida
